## How to Get Listed

If you've contributed to Promps and would like to be listed here:

1. **For GitHub users**: Your contribution will be automatically recognized through Pull Requests, Issues, or Discussions. Let us know if you'd like to be added to this list!

2. **For anonymous contributors**: If you prefer not to share your GitHub username, you can still be listed with just your name or a pseudonym.

3. **Contact us**:
   - Comment on your Issue/PR saying you'd like to be added
   - Email: promps-dev@zundou.org

### What Counts as a Contribution?

- [Bug] Reporting bugs
- [Idea] Suggesting features
- [Test] Testing and providing feedback
- [Note] Improving documentation
- [PC] Contributing code
- [Globe] Translating to other languages
- [Art] Design suggestions
- [Comment] Helping other users

## リストに掲載されるには

Promps に貢献し、このリストに掲載されたい場合：

1. **GitHub ユーザーの場合**: プルリクエスト、Issue、またはディスカッションを通じて貢献が自動的に認識されます。このリストに追加を希望する場合はお知らせください！

2. **匿名の貢献者の場合**: GitHub ユーザー名を共有したくない場合でも、名前や仮名だけでリストに掲載できます。

3. **連絡方法**:
   - Issue/PR にコメントして追加希望を伝える
   - メール: promps-dev@zundou.org

### 何が貢献とみなされますか？

- [Bug] バグの報告
- [Idea] 機能の提案
- [Test] テストとフィードバックの提供
- [Note] ドキュメントの改善
- [PC] コードの貢献
- [Globe] 他言語への翻訳
- [Art] デザインの提案
- [Comment] 他のユーザーへの支援
